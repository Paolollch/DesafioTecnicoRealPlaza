# Caso 2 - MoneyParts

Proyecto de consola en C# que genera todas las combinaciones posibles de denominaciones para representar un monto (múltiplos de 0.05 soles).

Cómo probar:

- Entrar a la carpeta publish y ejecutar el Caso2.exe generado.

Ejemplo de uso:

1. Ejecutar Caso2.exe.
2. Ingresar un monto como 0.5 o 10.50.
3. El programa mostrará hasta las primeras 100 combinaciones encontradas y el total.
